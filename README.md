# SmartCar-Assist
About O SmartCar Assist O AutoSense é um sistema embarcado para veículos que utiliza sensores para aumentar segurança e conforto. Um ultrassônico auxilia na ré com buzzer, o LDR acende os faróis automaticamente, o sensor de temperatura monitora o motor e o PIR detecta a aproximação do usuário para liberar as portas.
